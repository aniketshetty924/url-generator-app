Prospect URL Generator

Generate survey URLs for your campaign prospects by providing prospect details and a base URL template.

Single URL Generator
Multiple URLs Generator

How to use:

  • Provide a base URL template (e.g., https://surveys.adeptpanels.com/iws-gen-2/)
  • Select the environment (dev or uat)
  • Provide prospect details in JSON format
  • The tool will dynamically add parameters to the URL with values from the prospect data
  • Parameters mapped: isTest → environment, campaignDataId → prospectCampaignDataId, prospectEmail → workEmail, prospectName → firstName + lastName, prospectId → prospect.id
